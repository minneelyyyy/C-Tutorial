Comments

Comments are a very common element of programming languages. They are used to add information for programmers to read, they help with explaining the flow of the program, why some code exists, or to leave some details on something that should be done later on.

C has two types of comments, often called single-line comments and multi-line comments. 

// this is a single-line comment

/* this is a
multi-line comment */

Single-line comments are not available in C89. They were added in C99. Meanwhile multi-line comments have been in the language since it was created. So it is very common to see code use multi-line comments as single line comments like this: 

/* TODO: fix this code */
